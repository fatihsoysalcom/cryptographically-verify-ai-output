# Cryptographically Verify AI Output

This Python script demonstrates how to cryptographically verify the integrity of AI-generated output using hashing. It simulates generating AI text, creating a SHA-256 hash as a proof, and then verifying that proof against the original output.

## Language

`python`

## How to Run

Save the code as `verify_ai_output.py` and run it from your terminal using: `python verify_ai_output.py`
